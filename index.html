Projects

I make projects of all sorts, with a special interest in machine learning and artificial intelligence algorithms.

A2 Yum

Genetic algorithm solver for finding the shortest route to visit all restaurants in Ann Arbor.

Lemoneda

Anonymous social network for high school students. Developed in Python using Django & AngularJS.

CheckUp

Building management service I programmed during internship at Popcorn5, Summer of 2014.

Readly.IO

Article reminder service using Twitter as its interface. Developed in NodeJS.

Segment

Open-source clone of the IGF winning game, Perfection. Developed in C++ using Cocos2D-x.

GimmeGlasses

Weekend hack that places glasses on faces that users upload. Developed in Python & C++, using OpenCV.

WhiteBox

Weekend hack that attempts to answer user's questions. Developed in Python, using NLTK.

moon.chase.star

2D ball bouncing puzzle game. Developed in C++, using SFML and a custom game engine.

Round & Round

3D ball rolling puzzle game. Developed in C++, using Leadwerks 2.0 engine.
